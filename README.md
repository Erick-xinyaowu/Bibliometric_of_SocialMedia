# Bibliometric_of_SocialMedia

> 研究方案：注意力经济与信息扩散驱动的社交媒体演化机制研究——以抖音“生成式 AI”话题数据为例

## 1. 总理论框架：核心逻辑关联

本研究构建了一个**“资源 - 过程 - 形态”**的综合演化模型：

- 注意力经济 (Attention Economy) 作为宏观背景，定义了社交媒体用户注意力的稀缺性与竞争本质；
- 信息扩散 (Information Diffusion) 作为动态驱动力，描述了热度在节点间的流动与反馈过程；
- 演化机制则通过以下四个维度最终呈现：
  - 形态分布： 注意力在用户与内容上的非均衡分配（长尾理论）。
  - 语义内核： 吸引注意力的内容主题特征（LDA vs. BERTopic）。
  - 时间衰减： 注意力随时间老化的物理轨迹（老化测度）。
  - 关联分析： 互动深度与注意力留存的机制关联（扩散机理）。

## 2. 四大研究维度深度细化

### 维度一：基于长尾理论的内容与主体分布特征检验

- 研究内容： 分析评论文本及用户产出行为的分布规律。
- 学术目标：
  - 词频分布拟合特征： 检验评论关键词是否符合 Zipf 分布，揭示注意力在语义空间的聚焦度。
  - 互动热度分布检验： 分析单条评论获赞数是否呈现典型的长尾分布（幂律分布），揭示注意力强度的极化现象。
- 理论支撑： 利用长尾理论解释为何极少数“头部词汇”和“头部神评”占据了绝大部分注意力资源，而海量的“长尾内容”构成了话题基座。

### 维度二：语义内容特征分析：LDA 与 BERTopic 的对比检验

- 研究内容： 探索吸引注意力的内容本质。
- 技术路径： 对比基于统计概率的 LDA 与基于深度学习语义嵌入的 BERTopic 聚类效果。
- 分析要点：
  - 检验不同模型对短视频碎片化评论的聚类一致性。
  - 分析生成式 AI 话题下，哪些主题（如技术震撼、就业焦虑、创意工具）更容易获得高强度扩散。

### 维度三：注意力老化测度：衰减曲线拟合与半衰期分析

- 研究内容： 刻画注意力随时间失效的动态过程。
- 核心方法：
  - 多模型拟合： 对比指数衰减、幂律衰减与对数正态模型的拟合特征。
- 测度指标： 计算注意力半衰期 ($T_{1/2}$)（累计互动量达 50% 的时长）。
- 理论应用： 结合注意力经济理论，分析信息从“新鲜”到“老化”的客观节律，量化 Web 2.0 时代信息的“保鲜期”。

### 维度四：演化关联分析：基于信息扩散理论的机制关联探究

- 研究内容： 探究互动深度如何与注意力老化进程产生关联。
- 变量设置：
  - 扩散强度指标： 点赞数（like_count）。
  - 扩散深度指标： 二级评论率（sub_comment_count）。
- 分析方法： 检验二级评论带来的“深度扩散”与注意力衰减系数（老化速度）之间的相关性。
- 机理总结： 探讨社交互动是否能延缓信息老化，形成注意力的“二次唤醒”机制。

## 4. 详细技术路线图 (Technical Roadmap)

- 预处理： $\Delta T = T_{comment} - T_{video\_create}$（单位：小时）。
- 分布检验： 绘制 Log - Log 图，计算幂律指数 $\alpha$。
- 语义聚类： 分别运行 LDA 和 BERTopic，通过一致性得分 (Coherence) 对比效果。
- 曲线拟合：利用 scipy.optimize.curve_fit 拟合三种模型。根据 AIC/BIC 准则 判定最优衰减形态。
- 关联探究： 对“子话题类别”、“二级评论率”与“半衰期”进行 Spearman 相关性检验。
